Description List (dl) - HTML List Types

Unordered List (ul)
Used for grouping items where order is not important. Items are typically marked with bullets.
Ordered List (ol)
Used for sequential items. Items are automatically numbered.
Description List (dl)
Used for term-definition pairs. Contains dt (term) and dd (description) elements.
