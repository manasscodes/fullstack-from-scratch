Block Elements

Block elements take up the full width available and start on a new line.

This div is a block element
This section is also a block element

Inline Elements

Inline elements only take up as much width as necessary and don't force new lines. They flow with the text content.

Common Block Elements:

  • <div>
  • <p>
  • <h1> to <h6>
  • <section>
  • <article>

Common Inline Elements:

  • <span>
  • <a>
  • <strong>
  • <em>
  • <img>
